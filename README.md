# python-practice

このリポジトリは、Python を使った売上データの集計と可視化の練習用です。  
日別売上をグラフ化し、平均線・ランキング出力・合計/平均の算出まで行いました。

---

## 🧩 プロジェクト概要

Pythonを使って売上データを自動集計・可視化する練習プロジェクトです。  
日別・月別の売上グラフや、上位ランキングのCSV出力を自動生成します。

---

## 🔧 使用技術
- Python 3.9
- pandas
- matplotlib
- plotly
- GitHub Pages
